# How to install

* Clone this repository. `git clone https://github.com/lbotano/quchat-server.git`
* Go to the repository's directory and run `yarn install`.
* Create a file called `.env`.
* Inside that file write `PORT=<the port you want>`
* Run with `yarn start`.
